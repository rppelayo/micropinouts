8-bit RISC

  • Up to 20 MHz clock
  • 2.0V - 5.5V operating voltage

ESP32

Dual-Core WiFi

38-pin development board with dual-core processor, WiFi, and Bluetooth capabilities.

  • 34 GPIO pins

  • 4MB flash memory
  • 520KB RAM
