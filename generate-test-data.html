Generate Test Data

This page will help you populate the MicroPinouts database with sample pinout data for testing.

Setup Instructions

Option 1: Using PHP (Recommended)

  1. Install PHP and MySQL on your system
  2. Run the database setup: php setup-database.php
  3. Run the test data generator: php test-data.php

Option 2: Manual Database Setup

  1. Create a MySQL database named 'micropinouts'
  2. Run the SQL commands from database/schema.sql
  3. Use the API endpoints to create pinouts manually
